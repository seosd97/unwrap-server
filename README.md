# unwrap-server

GraphQL API server for [unwrap.fm](https://www.github.com/seosd97/unwrap.fm) - a music streaming analytics platform.

## Overview

A GraphQL server that processes Spotify listening history and provides analytics APIs for music insights and statistics.
